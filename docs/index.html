Features

container cards
Live status, uptime, CPU and RAM per container. Stop, restart, and start in one click.
system metrics
CPU, RAM, and disk usage bars — updated every 5 seconds.
update all
Pulls latest images for every container, restarts only those that changed.
no build step
Vanilla JS and CSS. Static files reload on save — no bundler, no framework.
detail view
Image, ports, mounts, networks, live stats, and deep link to Dozzle logs.
mobile friendly
Responsive layout — works on phones and tablets without a separate view.

Deploy

# docker-compose.yml
services:
  dashboard:
    build: ./app
    container_name: dashboard
    restart: unless-stopped
    ports:
      - "3001:8080"
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - ./app/static:/app/static:ro
      - ./data:/data
    environment:
      - DOZZLE_URL=http://your-host:9999
      - CONTAINER_NAME=dashboard
      - HOSTNAME_DISPLAY=station
